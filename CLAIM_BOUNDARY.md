# Claim Boundary

This repository contains a toy proof-of-architecture for a cognitive immune layer.

## Supported claim

In the tested Ulam boundary domain, separating intake from commit improved the control architecture:

```text
wide buffer
strict quorum commit barrier
portable receipt memory
```

The strongest result:

```text
wide intake + strict quorum protected the core better than porous direct admission
```

## Not claimed

This repo does not prove:

- consciousness;
- general AI alignment;
- human persuasion theory;
- political theory;
- legal truth;
- institutional legitimacy;
- universal security;
- a guaranteed zero false-block rate in real systems.

## Real-world translation

Use this as an architecture pattern, not a metaphysical proof.

The pattern:

```text
inspect widely
commit carefully
record crossings
audit updates
```

## Engineering caution

A quorum is only as good as witness independence.

A cryptographic receipt can prove provenance, order, and integrity.

It cannot prove that a claim is true or that the signer was honest.

A signed lie is still a lie.

The receipt gives the receiver something to audit before allowing a signal to alter protected state.
